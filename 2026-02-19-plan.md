# plan.md — 鄭南榕事件互動體驗網站 開發計畫

## 🎯 專案定位

### 一句話描述
一個讓民眾「體感」戒嚴後言論恐怖的沉浸式網站——透過5000頁監控檔案和「唯一死刑」法條，讓訪客理解：解嚴之後，發表文字仍然可以被判死刑。

### 核心訴求（優先序）
1. **「二條一」的恐怖** — 讀一本書、刊一篇文，唯一刑罰就是死刑，法官無從輕量刑
2. **監控的無所不在** — 5000頁檔案不是數字，是每天被跟蹤、被記錄的日常
3. **解嚴 ≠ 自由** — 1987年解嚴，1989年仍以叛亂罪追訴出版者
4. **一個人的抉擇** — 71天自囚，從收到傳票到最後一刻的心路

### 與陳文成案網站的關係（~/projects/Chen-Wen-chen/）
- **共享設計系統：** 「檔案恐怖」(Bureaucratic Horror) 視覺語言
- **共享技術棧：** React + TypeScript + Tailwind + Vite
- **共享像素風基礎設施（可直接複製）：**
  - `src/components/pixel/PixelScene.tsx` — 共用 wrapper（viewport lazy loading、responsive scaling、reduced-motion 降級）
  - `src/components/pixel/PixiCanvas.tsx` — PixiJS v8 Canvas renderer
  - `src/components/pixel/usePixiApp.ts` — React hook for PixiJS Application
  - 15 個程序化場景（每個 6K–12K 行 PixiJS Graphics）可作為繪製參考
  - 標準：320×180 base resolution、`image-rendering: pixelated`、scroll progress 驅動動畫
- **獨特元素：** 燒焦邊緣 (burnt edges)、火焰漸層、71天倒數、法條互動
- **可共用元件：** 紅acted文字、解密印章、政府檔案樣式、scroll reveal hooks

### Chapter 4: 監控真相 — 5000頁的凝視
**時長：** 4-5個螢幕高度
**設計目標：** 全站恐怖感最高點——讓訪客「溺」在監控檔案中

**內容：**

#### 4a. 監控的規模
- **像素風場景重現：情報辦公室**（鳥瞰情治人員伏案撰寫報告、檔案櫃、轉盤電話、菸灰缸）
- 數字衝擊：「情治單位對鄭南榕的監控檔案超過5000頁」
- 視覺：5000張紙張疊起的高度動畫（比一個人還高）
- 換算：如果每天讀10頁，要讀500天，超過一年

#### 4b. 監控檔案瀏覽器
- **核心互動元素：** 模擬檔案櫃抽屜
  - 拉開抽屜 → 文件堆疊排列
  - 每份文件可點擊展開
  - 文件格式：情治機關表頭、日期、案號
  - 紅acted遮蔽：點擊黑條 →「解密」印章蓋下 → 文字顯現
  - 多機關來源：國安局、調查局、警備總部（不同表頭樣式）

#### 4c. 被監控的細節
- 分類展示監控內容：
  - 行蹤監控（每日動態）
  - 來訪者紀錄
  - 電話監聽
  - 出版物內容審查
  - 社交關係圖譜
- 每個類別配3-5份原始檔案摘錄

#### 4d. 荒謬與恐怖
- **🖥️ CRT迷你互動：監控報告填寫**（~30秒）
  - CRT終端機開機動畫 → 情報報告表格
  - 根據青谷專案實際格式：選擇來源可靠性（甲/乙/丙）、內容正確性（一/二/三）
  - 從選項中選擇監控對象的日常動態摘要（皆為平凡瑣事）
  - 選擇建議處置：繼續監控 / 加強監控 / 報請上級
  - 提交 → 報告歸檔 → 計數器：「已建檔 1/5,000 份」
  - 結語：「這只是5000頁中的一頁。每一頁的另一端，都是一個人的日常。」
- 監控中的荒謬細節（吃什麼、見了誰、說了什麼瑣事）
- 對比：冷冰冰的公文語言 vs. 一個活生生的人
- 持續瀏覽計數器：「您已閱覽第 ___/5000 頁」
- **解密互動**（監控報告文件）：點擊黑條揭露監控內容
  - 範例：對象於██前往██ → 對象於09:15前往印刷廠；██委員會電話██ → 治喪委員會電話遭干擾

## ⚠️ 敏感內容處理原則

1. **自焚場景：** 絕不呈現圖形化暴力畫面。以環境氛圍（火光、溫度漸層色調）和時間軸推進暗示，不描繪身體細節。
2. **遺言引用：** 完整引用，提供上下文。這是政治宣言，不是自殺遺書——框架很重要。
3. **家屬隱私：** 葉菊蘭等家屬相關內容，僅引用公開聲明和已出版回憶。
4. **政治中立：** 呈現事實和政府文件，不做政黨攻防。重點是制度性問題，不是個別政治人物。
5. **內容警告：** 網站入口處提供內容警告，說明涉及政治迫害和自焚事件。
